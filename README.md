# Minesweeper

A progressive-difficulty Minesweeper game built with vanilla HTML, CSS, and JavaScript.

## How to Play

- **Left click** to reveal a cell
- **Right click** to place/remove a flag
- Clear all non-mine cells to advance to the next level
- Each level increases grid size and mine count

## Levels

The game features 10 built-in levels that scale from a beginner-friendly 5x5 grid up to a challenging 16x16 board. Beyond level 10, difficulty continues to increase procedurally.

## Running Locally

Just open `index.html` in any modern browser — no build step or server required.

## Features

- 10+ progressive levels
- Timer and flag counter
- Level select menu with progress tracking
- Progress saved to localStorage
- Responsive design (works on mobile)
- First-click safety (you'll never hit a mine on your first click)
